#### 元组

元组是可以不同质的数据结构，使用[()]()表示空元组。不存在单个元素的元组。

##### 元组比较

如果两个元组的大小相等，且元素的类型一一对应，那么这两个元组可以相互比较。

##### 元组操作函数

###### fst

获取大小为2的元组的第一个元素。

###### snd

获取大小为2的元组的第二个元素。

###### zip

将两个列表压缩为一个元组列表，且会切断较小列表。

```haskell
main = do
  print $ zip [1, 2] ['a', 'b', 'c'] -- [(1,'a'),(2,'b')]
  print $ zip [1 ..] ['a', 'b', 'c'] -- [(1,'a'),(2,'b'),(3,'c')]
```

##### 访问多元组

标准并没有提供访问多元组分量的方式，需要手动实现：

```haskell
first (a, _, _) = a
second (_, b, _) = b
third (_, _, c) = c

main = do
  print $ first (1, 2, 3) -- 1
  print $ second (1, 2, 3) -- 2
  print $ third (1, 2, 3) -- 3
```
