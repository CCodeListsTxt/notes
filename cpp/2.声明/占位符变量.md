c++26 引入。

当 `_` 作为变量声明、非静态类成员、lambda 捕获、结构化绑定时，其隐式包含了 `[[maybe_unused]]` 属性。

```cpp
[[nodiscard]]
auto foo() -> int { return 0; }

auto main() -> int {
    foo();              // warning: ignoring return value
    auto _ = foo();     // equal to [[maybe_unused]] auto _ = foo();
    return 0;
}
```
