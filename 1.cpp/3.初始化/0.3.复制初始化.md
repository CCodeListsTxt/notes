从另一个对象初始化对象时，发生[复制初始化]()[^1]。

###### 效果

初始化类类型时：

* 如果初始化器是[纯右值表达式]()，且其无cv限定的类型和『目标类型』相同，进行[复制消除]()。
* 





# 示例

###### 示例1

```cpp
struct T {
	T() { std::println("construct\n"); }
	T(const T&) { std::println("copy\n"); }
	~T() { std::println("destruct\n"); }
};

auto main() -> int {
	try {
		throw T{};	// construct
	}
	catch (T t) {	// copy

	}	// destruct x 2

	return 0;
}
```





[^1]:捕获[按值抛出]()的异常时，也进行复制初始化。[示例](#示例1)
